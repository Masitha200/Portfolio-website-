AetherPDF

Standalone PDF Page Organizer, Rotator & Editor

AetherPDF is a standalone, lightweight system application built to solve common document management and page organization headaches offline. It combines a visual drag-and-drop page workspace (for page reordering, duplication, deletion, and rotation) with an interactive workspace that allows you to manage PDF documents locally and securely.

Add & Merge PDF Files

Insert new pages from external files or merge multiple separate PDF documents into a single, cohesive file output locally.

Reorder & Sequence Pages

Arrange document pages in any order you want with real-time drag-and-drop visual page thumbnail feedback.

Rotate, Duplicate & Delete

Quickly rotate skewed or scanned pages by 90° clockwise, remove invalid layout sheets, or duplicate key pages instantly.

100% Offline Client Execution

No cloud server connections or REST callbacks required. The entire workflow works locally on the desktop to guarantee 100% data security.

Windows SmartScreen Note (False Positive)

When launching the software for the first time, Windows Defender SmartScreen might show a warning message saying "Windows protected your PC". This is 100% normal and is not a virus or malware.

Why does this happen?

This happens because AetherPDF is a newly compiled standalone desktop application built using Electron and React that is not signed with an expensive Microsoft Developer Certificate. As a result, Windows reputation-based protection flags it temporarily until more users download and run it.

How to run the software safely:

  1. Double-click the downloaded AetherPDF.1.0.0.exe file.
  2. In the blue popup window, click on the "More info" link under the text.
  3. A new button called "Run anyway" will appear at the bottom right.
  4. Click "Run anyway" to launch the application safely.
