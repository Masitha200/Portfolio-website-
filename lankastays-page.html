Web Application Showcase

LankaStay

Discover Sri Lanka's Finest Stays

LankaStay is a premium web booking and reservation client platform designed to simplify trip planning and hotel finding across Sri Lanka. Built with responsive interfaces, verified property inventories, booking summary builders, and localized travel logs, the system connects travelers directly with luxury seaside resorts and mist-covered mountain bungalows.

Concept Project: LankaStay is a high-fidelity frontend demo and design showcase. It is a concept/dummy site built to demonstrate interface layouts, booking flow behaviors, and local tourism components.

Verified Property Inventory

Lists hand-inspected rooms, beachfront villas, and eco-friendly retreats with detailed pricing, review logs, and free cancellations.

Destination-Based Querying

Quick-link filtering structures designed to showcase popular places such as Colombo, Kandy, Galle, and Ella instantly.

Secure Booking Summaries

Transparent reservation logs featuring cancellation options, date-duration checks, and instant reservation receipts.

Fully Responsive Layout

Optimized using styling components for mobile-drawer menus, ensuring users have access to bookings on their mobile devices during active trips.
